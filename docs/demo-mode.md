# Demo 안내 모드 명세 - v1.0

본 문서는 시연(Demo) 시 사용될 안내 모드의 명세를 정의합니다.

## 1. 목적

- 시연 참가자(청중)에게 데모의 목적과 사용 환경(Sepolia)을 명확히 안내한다.
- FAQ를 제공하여 원활한 시연을 돕는다.

## 2. 트리거

- 사용자가 메인 페이지(`/`)에 처음 진입 시 1회 노출된다.
- (구현: `localStorage`에 방문 기록이 없으면 모달을 띄운다.)

## 3. 모달 내용 (UI)

- 제목: ZKP 투표 데모 안내

  - "MetaMask 지갑 연결 및 Sepolia 네트워크 설정이 필요합니다."
  - "테스트용 Sepolia ETH가 필요하면 운영진에게 문의해주세요."
